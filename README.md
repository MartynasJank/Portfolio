# martybuilds.dev

Personal portfolio site. No frameworks, no build step. Just HTML, CSS, and vanilla JS.

## Stack

- Pure HTML / CSS / vanilla JS
- Zero dependencies
- Works by opening `index.html` directly in a browser

## Features

- Custom cursor with lag effect
- Letter scramble on name hover
- Subtle CSS glitch animation
- Scroll reveal on sections
- Interactive terminal (`whoami`, `ls projects/`, `cat skills.txt`, `help`, `clear`)
- Project cards with scanline hover effect
- Konami code easter egg (↑↑↓↓←→←→BA)
- Responsive, mobile-friendly
- Respects `prefers-reduced-motion`

## Running locally

No server needed. Clone the repo and open `index.html` in a browser.

```bash
git clone https://github.com/MartynasJank/Portfolio.git
cd Portfolio
# open index.html
```
